Aplikasi terdapat 3 Menu.
- Home : dapat mengganti tampilan menjadi list atau grid 2 column, jika di klik itemnya, akan menampilkan detail produk. Detail produk berisikan informasi detail produk dengan image carrousel.
- Admin : jika slide ke kanan, maka pada bagian sebelah kiri item akan ada button edit untuk mengubah informasi produk, jika slide ke kiri, di sebelah kanan akan terdapat button trash untuk menghapus data yang ada. Akan muncul prompt dahulu untuk meyakinkan apa benar-benar data ingin di delete.

This application is about A store product list.
Who is the target or this app ? An Admin of the store only.
Why? To control all the product that the store sell.
This application menu includes :
- Home : Base display, to show the product list. There are 2 type of showing the product, by grid 2 column or 1 list. If the item is clicked, it will show the product detail. The detail contains the information of product and some of the product's pictures that shows by image carrousel.
- Admin : This menu can be accessed by slide to right. In the left of product item, there is an edit button to change the product detail information. If admin slide to right then there is trash button to delete the product. If user choose to delete product, there will be a prompt to show if user really want to delete the product.
- About : To display the app creator information.
