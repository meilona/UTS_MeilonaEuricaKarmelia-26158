Aplikasi terdapat 3 Menu.
- Home : dapat mengganti tampilan menjadi list atau grid 2 column, jika di klik itemnya, akan menampilkan detail produk. Detail produk berisikan informasi detail produk dengan image carrousel.
- Admin : jika slide ke kanan, maka pada bagian sebelah kiri item akan ada button edit untuk mengubah informasi produk, jika slide ke kiri, di sebelah kanan akan terdapat burron trash untuk menghapus data yang ada. Akan muncul prompt dahulu untuk meyakinkan apa benar-benar data ingin di delete.
